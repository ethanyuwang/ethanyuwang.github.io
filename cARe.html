cARe

AR interface for an existing healthcare robot, which uses machine learning to automatically detect people and monitors, and provides context sensitive actions for more intuitive robot operation.

FEATURES

Zoom

Zoom improves upon manual zoom by using an object’s relative size to determine the appropriate zoom magnitude.

Approach

Approach provides an automated drive method, by facing the object and moving in a straight line towards it.

Center

Center allows the doctor to focus the camera on a object, by using the target’s relative coordinates in the image.

Identify

Identify allows the doctor to access information on a staﬀ member or patient, such as specialty or medical history.
